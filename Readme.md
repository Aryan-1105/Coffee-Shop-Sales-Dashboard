# ☕ Coffee Shop Sales Dashboard

## Brief Summary
An interactive Excel dashboard that analyzes coffee shop sales data to uncover trends, customer behavior, and actionable business insights.

---

## Overview
This project focuses on analyzing coffee shop sales data using Microsoft Excel and presenting insights through an interactive dashboard.  
The dashboard visualizes key metrics such as total sales, customer footfall, hourly sales trends, product performance, category distribution, and store-wise comparisons.  

The project demonstrates practical data analysis, visualization, and dashboard storytelling skills using Excel.

---

## Problem Statement
Coffee shop management requires a clear and interactive way to understand sales performance, peak business hours, product demand, and store-wise performance.  
Raw sales data alone does not provide meaningful insights, making it difficult to optimize staffing, inventory, and marketing strategies.

---

## Dataset
The dataset consists of transactional sales data from multiple coffee shop locations and includes:
- Order date and time  
- Store location  
- Product name  
- Product category  
- Product size  
- Quantity sold  
- Sales amount  

The data was cleaned and prepared in Excel before analysis.

---

## Tools and Technologies
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Slicers (Day and Month filters)  
- Advanced Excel formulas  
- Conditional formatting  
- Data visualization techniques  

---

## Methods
- Data cleaning and formatting  
- Creation of calculated fields for KPIs  
- Aggregation of data using pivot tables  
- Visualization using bar charts, line charts, and pie charts  
- Interactive filtering using slicers  
- Dashboard layout and design for business storytelling  

---

## Key Insights
- Peak sales occur between **8–10 AM**, with the highest activity at **9 AM**
- Tea products contribute the highest revenue (**39%**), outperforming coffee
- Espresso and tea-based beverages are the top-selling products
- Customer size preference is evenly distributed across Small, Regular, and Large
- All store locations show balanced sales and footfall performance
- Weekdays perform better than weekends, with Saturday recording the lowest sales  

---

## Dashboard / Model / Output
The final output is an **interactive Excel dashboard** that displays:
- Total sales, customer footfall, and average order metrics  
- Hourly sales volume  
- Top 5 products by revenue  
- Category-wise and size-wise sales distribution  
- Store location performance comparison  
- Sales trends across weekdays  

The dashboard supports interactive analysis using **day and month slicers**.

---

## How to Run this Project?
1. Download the Excel dashboard file from the repository  
2. Open the file using Microsoft Excel (desktop version recommended)  
3. Enable editing if prompted  
4. Use slicers to filter data by day or month  
5. Interact with charts and KPIs to explore insights  

---

## Results & Conclusion
The Coffee Shop Sales Dashboard successfully converts raw sales data into meaningful insights.  
It helps identify peak business hours, high-performing products, and sales patterns across locations, supporting data-driven decisions for staffing, inventory management, and marketing strategies.
